# Class 05: CSS

## CSS là gì

- CSS dùng để tuỳ chỉnh các phần tử của trang web

## CSS rules

```css
selector {
  // css rules
  color: red;
}
```

- Selector: quy định những phần tử nào được áp dụng luật CSS
  - Tên thẻ: p, b, a : để xác định nhóm thẻ được áp dụng luật
  - #id: xác định id của một phần tử

```css
p {
  color: red;
}
```
